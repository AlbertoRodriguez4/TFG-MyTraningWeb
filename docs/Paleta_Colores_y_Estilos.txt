PALETA DE COLORES Y ESTILOS - THE TRAINING HUB
================================================

1. FRAMEWORKS Y HERRAMIENTAS DE ESTILO
---------------------------------------
- Vuetify 3: Framework principal de componentes UI para Vue 3.
- Sass: Preprocesador CSS utilizado en el proyecto.
- Material Design Icons (MDI): Libreria de iconografia a traves de @mdi/font.
- CSS personalizado: Gestion de variables globales y temas en src/styles/global.css.
- Tailwind-like palette: Uso extensivo de colores de la escala Slate en componentes personalizados.

2. SISTEMA DE TEMAS
-------------------
El proyecto implementa un sistema de temas gestionado mediante Pinia (useTheme.ts) con soporte para tres modos:
- Dark (Oscuro): Tema por defecto.
- Light (Claro): Tema alternativo.
- Auto: Detecta automaticamente las preferencias del sistema del usuario.

El tema se persiste en localStorage y se aplica mediante clases CSS (dark/light) y el atributo data-theme sobre el elemento <html>.

4. COLORES SEMANTICOS Y DE ESTADO
----------------------------------
A lo largo de los componentes se utilizan colores especificos para transmitir informacion:

- Amarillo/Dorado: #fbbf24, #FFD700, #f59e0b
  Uso: Estadisticas, monedas, niveles, recompensas, estrellas.

- Azul/Cian: #38bdf8, #00D2FF, #3b82f6, #7dd3fc
  Uso: Informacion, estadisticas de progreso, roles de agua, elementos interactivos.

- Verde/Esmeralda: #34d399, #4ADE80, #22c55e, #86efac
  Uso: Exito, miembros conectados, disponibilidad, confirmaciones.

- Purpura/Violeta: #a78bfa, #8B5CF6, #A855F7, #c4b5fd, #e9d5ff
  Uso: Roles de administrador, rareza de objetos, niveles especiales, interfaz de edicion de usuario.

- Rojo/Naranja: #ef4444, #e85d26, #f97316, #f87171, #fca5a5
  Uso: Errores, advertencias, eliminacion de elementos, acentos en chat, peligro.

- Escala de Grises (Slate): #0f172a, #475569, #64748b, #94a3b8, #cbd5e1, #e2e8f0, #f8fafc
  Uso: Textos secundarios, fondos de paneles, bordes, iconos inactivos, sombras y jerarquias visuales.

6. ARQUITECTURA Y PATRONES DE ESTILO
-------------------------------------
- Variables CSS Globales: Centralizacion de colores y fondos en global.css.
- Scoped Styles: La mayoria de estilos se definen dentro de los componentes .vue para evitar colisiones.
- Sobrescritura de Vuetify: Se sobreescriben clases como .v-application y .v-card para integrar el tema personalizado.
- Transparencias: Uso extensivo de rgba para bordes, fondos de tarjetas y textos secundarios, permitiendo adaptacion suave entre temas.
- Efectos Visuales:
  * Glow: Efectos de resplandor en estadisticas (box-shadow con colores dinamicos).
  * Animaciones: fadeSlideUp para aparicion de elementos en chat.
  * Gradientes sutiles y sombras para crear elevacion en tarjetas.

7. RESUMEN DE IDENTIDAD VISUAL
-------------------------------
The Training Hub utiliza una identidad visual moderna y gamificada:
- Base oscura dominante (Dark Mode) que evoca interfaces de dashboards deportivos y videojuegos.
- Acento energico en amarillo/dorado (#ffcc00) que simboliza energia, recompensas y motivacion.
- Codificacion de colores semantica clara para estados, roles y niveles de dificultad.
- Modularidad: Ciertos modulos como el Chat adoptan paletas propias para diferenciar la experiencia de usuario.
- Enfoque en la accesibilidad con contrastes definidos entre textos primarios, secundarios y fondos.
