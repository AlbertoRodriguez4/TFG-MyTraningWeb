//En que consiste thetraininghub: 

tengo una web llamada TheTrainingHub que es una plataforma web social-gamificada pensada para aumentar la adherencia a entrenar en gimnasio mediante quedadas (salas) y retos. Los usuarios crean/unen a salas para entrenar en grupo, aceptan retos, hacen check-ins y reciben recompensa en forma de XP, monedas, dinero virtual y atributos. El progreso sube de nivel y desbloquea acceso a salas con requisitos; la economía interna y el sistema de atributos permiten crear un ciclo motivador (social + recompensa).
